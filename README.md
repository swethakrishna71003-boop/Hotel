# Hotel Booking System 🏨

## 📌 Description
This is a full-stack Hotel Booking System where users can register, login, view rooms, and book rooms. The system uses secure authentication with JWT and stores data in MySQL.

---

## 🚀 Features
- User Registration & Login
- Secure Authentication (JWT)
- Password Hashing (bcrypt)
- View Available Rooms
- Book Rooms
- Cancel Booking
- Booking History
- Protected Dashboard

---

### Frontend

### Backend

## ▶️ How to Run

1. Clone the repository
    https://github.com/swethakrishna71003-boop/Hotel.git
2. Install dependencies
   npm install
3. Start server
   node server.js
4. Open in browser
   http://localhost:4000

---

## 🔐 Authentication Flow
- User logs in
- Server generates JWT token
- Token stored in browser
- Protected routes require token
